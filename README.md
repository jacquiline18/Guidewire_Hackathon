# InsurIntel AI – Intelligent Insurance Platform

InsurIntel AI is an AI-powered parametric insurance prototype designed to protect gig delivery workers from income loss caused by external disruptions such as heavy rain, extreme weather, pollution alerts, or service outages.

The system automatically detects disruptions using real-time data and processes claims without manual requests.

This project was built for Hackathon 2026.

--------------------------------------------------

## Problem Statement

Gig workers such as delivery riders depend on daily income.

During situations like:
- Heavy rain
- Floods
- Extreme heat
- Pollution alerts
- App outages

they cannot work and lose earnings.

Currently, there is no automatic insurance system that protects their income.

InsurIntel AI solves this problem using automated, AI-based insurance.

--------------------------------------------------

## Solution

InsurIntel AI provides:

- Weekly income protection insurance
- Real-time disruption detection
- Automated claim processing
- Fraud detection validation
- Predictive weather alerts

The system ensures riders receive compensation only when valid disruptions occur.

--------------------------------------------------

## Features

### 1. Rider Registration
Rider enters name, city, and income.

### 2. Weekly Insurance Plan
Rider can activate weekly coverage.

### 3. AI Risk Assessment
Premium decided based on disruption risk.

### 4. Predictive Disruption Alerts
Weather API used to warn rider before events.

### 5. Automatic Claim System
Claims triggered when disruption detected.

### 6. Fraud Detection
- Insurance active check
- Location mismatch check
- Duplicate claim check

### 7. Wallet Simulation
Approved claims are added to rider wallet.
